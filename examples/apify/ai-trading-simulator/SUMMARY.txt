CORE FEATURES IMPLEMENTED:

1. Market Data Generation
   - Neural-inspired OHLCV synthesis
   - Configurable volatility (0.01-0.5)
   - 4 trend types (bullish, bearish, sideways, volatile)
   - 6 timeframes (1m, 5m, 15m, 1h, 4h, 1d)
   - Seeded randomization for reproducibility

2. Technical Indicators
   - SMA (Simple Moving Average)
   - EMA (Exponential Moving Average)
   - RSI (Relative Strength Index)
   - MACD (Moving Average Convergence Divergence)
   - Bollinger Bands (volatility bands)

3. Trading Strategies
   - Mean Reversion (Bollinger Bands-based)
   - Momentum (MACD crossover)
   - Breakout (Support/resistance levels)
   - ML Neural (Multi-indicator weighted)

4. Backtesting Engine
   - Trade execution simulation
   - Position management
   - P&L calculation
   - Comprehensive metrics:
     * Total return (%)
     * Sharpe Ratio (risk-adjusted returns)
     * Sortino Ratio (downside deviation)
     * Max Drawdown (worst decline)
     * Win Rate (profitable trades %)

5. Advanced Features
   - Order book simulation (Level 2 data)
   - News sentiment generation
   - Real-time streaming mode
   - Multiple export formats (JSON, CSV, Parquet)

  - mode: simulate | backtest | train | stream

  - timeframe: 1m, 5m, 15m, 1h, 4h, 1d
  - dataPoints: 10-100000
  - volatility: 0.01-0.5
  - trend: bullish, bearish, sideways, volatile
